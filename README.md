## Outils nécessaires

Pour suivre ce cours, installez les outils Java suivants sur votre machine :

### 1. Java Development Kit (JDK)

Vous pouvez télécharger la dernière version du JDK à partir de ces liens officiels :

- Site officiel Oracle :  
  https://www.oracle.com/java/technologies/downloads/

- (Alternative) Adoptium Temurin (JDK gratuit, recommandé pour les étudiants) :  
  https://adoptium.net/

> Choisissez une version **JDK 17 ou supérieure**, pour **Windows x64**, puis installez-la normalement.

### 2. IntelliJ IDEA (IDE Java)

Pour développer vos projets Java, nous utilisons **IntelliJ IDEA Community Edition** (gratuit) :

- IntelliJ IDEA Community Edition :  
  https://www.jetbrains.com/idea/download/

> Téléchargez la version **Community**, puis installez-la.  
> Pendant le cours, nous configurerons ensemble votre premier projet Java.
